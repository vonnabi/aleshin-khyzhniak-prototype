# Збережені анімаційні концепції

## Анімація №2

Альтернативна атмосферна анімація внутрішніх сторінок: світлові орбіти, частинки, велика фонова монограма `A · K · P`, золоті кола та scroll-reveal.

Файли:

- `animation-2.css` — візуальний шар;
- `animation-2.js` — генератор canvas-анімації.

Основний сайт зараз використовує анімацію головної сторінки. Цей варіант збережено для майбутніх промосторінок або повторного використання.
